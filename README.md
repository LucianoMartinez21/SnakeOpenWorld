# Open World Snake Game
Small Snake Game inside of an open world, Implemented using the public library raylib for C++.

Things to fix/Add are:
* Implement Audio, Music and SFX.
* Improvement in code and performance.

## How run it?
Run the following commands (for linux)
```
make
./app
```

## Tools Used
* Raylib: To develop the project using C.
* Firealpaca.
> Recommendation: to understand raylib use the quickstart (https://github.com/raylib-extras/raylib-quickstart.git)
